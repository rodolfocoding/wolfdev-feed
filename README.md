# Wolfdev Feed

Wolfdev feed é um projeto que simula o feed de uma rede social.

## Como utilizar

1. Clonar repositório

2. Instalar dependências
``npm i``

3. Iniciar projeto Local
``npm run dev``

4. Acessar aplicação
``http://localhost:5173/``

  <li>Vite</li><br>

</ul>
